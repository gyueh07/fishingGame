# 심해의 낚시꾼 UI 개편판

기존 텍스트 낚시게임의 저장 데이터와 게임 규칙을 유지하면서 화면과 파일 구조를 개편한 버전입니다.

## 이번 버전에서 바뀐 점

- 모바일과 PC에 맞춰 자동으로 배치가 바뀌는 반응형 게임 UI
- 낚시터, 양동이, 도감, 보스전, 성장, 광장 화면
- 물고기별 이미지를 사용하지 않고 종류별 아이콘과 등급 효과를 조합한 카드 UI
- 기존 명령어와 상세 전투 로그를 사용할 수 있는 게임 기록 패널
- 보스 처치 시 전용 코어 또는 재료 1~10개 지급
- 원본 물고기 능력치, 별, 훈련소 효과, 보스 수치와 전투 패턴 유지

## 파일 구조

```text
index.html                 화면 구조
css/legacy.css             기존 로그·모달 기본 스타일
css/game-ui.css            새 반응형 게임 UI
js/data/fish-data.js       물고기, 등급, 크기, 특성 데이터
js/data/combat-data.js     전투 범위, 보스, 코어 데이터
js/core.js                 계정, 저장, 공통 상태, 연구·훈련
js/combat.js               보스전과 PVP 전투
js/gameplay.js             낚시, 경제, 양동이, 도감 등
js/commands.js             기존 명령어 연결과 시작 처리
js/ui.js                   새 화면 표시와 버튼 연결
```

## 배포

프로젝트의 파일과 폴더 구조를 유지한 채 정적 웹 호스팅의 루트에 업로드합니다. 기존 게임과 같은 Firebase 프로젝트를 사용하므로 Firebase 설정과 허용 도메인은 기존 배포 환경을 그대로 사용합니다.

## 저장 데이터 호환

전투 능력치 버전은 원본과 동일하게 유지했습니다. 저장된 물고기의 `combat` 데이터가 다시 생성되거나 약해지지 않습니다.
